What this list does not claim

These are estimates of observable giving — the narrowest honest measure. Research across IPS, Boston College's Forum on Philanthropy, and the National Philanthropic Trust suggests observable figures capture 40–70% of true lifetime giving. A high tier means we cannot find evidence of generosity at scale — not that none exists.

Corrections welcome

Every Tier A and Tier B subject is contacted before publication and given a one-week window to respond. Responses are published verbatim on individual profiles. If you can document giving we have missed — with a source we can link — we will update the record and acknowledge it.

Submit via GitHub issues (preferred) or open a pull request against the profile JSON. Please include the citation URL.
