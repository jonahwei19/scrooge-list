# Scrooge List audit synthesis (Ralph iter 1)

Synthesized from 4 parallel sub-agents: cold-reader (index), profile-reader, plain-language, designer.

## Severity 1 — comprehension blockers (must fix)

1. **The 5% benchmark is unjustified inline on the index.** It's the load-bearing assumption (every shortfall, ratio, tier comes from it) but stated only once in the chart subhead, and the source/why is in methodology. Cold reader's exact words: "a skeptical journalist's first question is 'says who?' and the page doesn't answer it." Add an inline gloss anchoring the 5% to the IRS foundation payout floor.

2. **The deck reads as defending cautious givers.** "the more you give, the more likely you are to make a mistake, get criticized, and give less" — chain reads as "be cautious." Then the page criticizes non-givers. Cold reader: "Whose side is this on? The two clauses fight each other." NOTE: Jonah wrote this verbatim. Do not change without his sign-off — flag for next conversation.

3. **Profile role pills are jargon-as-pill-text:** `GRANT_OUT`, `TRANSFER_IN`, `PLEDGE`, `DIRECT_GIFT`. Plain-English replacements: "Grant", "Transfer in", "Pledge", "Direct gift". Same applies to the timeline legend.

4. **Acronyms never expanded inline on first use:** DAF, LLC, 990, Form 4, IPS, NBER, NPT, ICIJ, FEC, SEC, EIN, 501(c)(3), CZI. Index filter buttons say "DAF Heavy" and "LLC Opaque" — both inscrutable to lay readers.

5. **"$984B Undisbursed Shortfall · Tier A–C" stat strip** is the most prominent number on the page but reads as either annual or lifetime — ambiguous. Replace label with explicit unit/scope.

## Severity 3 — typography / spacing

- **Profile h1 + methodology h1 use opsz 144 at ~3.6rem** — letters look pinched. Use `opsz 72`.
- **Profile h2 spacing 72px** — too much. Reduce to 48px.
- **Wordmark article SOFT 80 vs wordmark SOFT 40** — clash. Match SOFT.
- **Cibar `.obs` width too small** when observable is < 1% of capacity (e.g. Ellison) — reads as "all hidden". Set `min-width:6px` on `.cibar-track .obs`.
- **Index ledger row at 800px**: "Stephen Schwarzman" / "Miriam Adelson" truncate. Bump first column 120 → 140px.
- **Methodology dropcap broken** when applied to wrong paragraph; verify scope.

## Severity 4 — dark mode

- Profile `flag-code` contrast ratio ~3.4:1 (fails WCAG AA). Switch text to `--rust` (lighter).
- Methodology dropcap "T" in saturated rust against dark paper — drop to `--gilt`.
- Profile `.vehicle.closed` opacity .75 too dim. Drop opacity to .9.
- Index dark ledger track barely distinguishable from body — bump `--paper-shadow` dark to `#4A3D27`.
